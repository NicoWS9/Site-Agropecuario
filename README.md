# Site-Agropecuario

🍃🌿Site sobre as novidades e automações da atividade agropecuária voltada para a agricultura familiar 

    - Site contendo uma página inicial ambientando o usuário o que é este site
    - Uma página msotrando os produtos oferecidos
    - Contato em caso de duvidas ou cadastros
    - Interface responsiva, aplicando conceitos de hierarquia visual
    - Interface intuitiva
    - Paleta de cores simbolizando a natureza e robustez
    -Framework Bootstrap v5.3 utilizada
